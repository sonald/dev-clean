Built for safe, confident cleanup

        

          

            

            
Fast scanning

            
Parallel traversal and streaming size calculation surface candidates quickly.

          

          

            

            
Rule attribution

            
Every finding can explain the built-in, custom, gitignore, or heuristic rule.

          

          

            

            
Risk and confidence

            
Default filters keep higher-risk dependency and gitignore discoveries hidden.

          

          

            

            
Plan before action

            
Generate machine-readable plans, review them, and apply only what you approve.

          

          

            

            
Undoable trash

            
Move targets into batch trash and restore the latest run when needed.

          

          

            

            
Terminal first

            
Use fast CLI flows or a keyboard-driven TUI when selecting targets manually.

          

        

      


      

        
A simple, safe workflow

        

          

            1
            
Scan

            dev-cleaner scan --explain
            
Find cleanable targets and see the exact rule behind each path.

          

          

            2
            
Plan

            dev-cleaner plan -o plan.json
            
Review the complete cleanup plan before touching the filesystem.

          

          

            3
            
Trash

            dev-cleaner apply plan.json --trash
            
Move approved directories into an undoable trash batch.

          

          

            4
            
Undo

            dev-cleaner undo
            
Restore the last trash batch if a cleanup needs to be reversed.

          

        

      


      

        

          
Safety by design

          

            
            

              
Conservative by default

              
Built-in rules target common build artifacts, while high-risk discoveries require opt-in.

            

          

          

            
            

              
Provenance you can trust

              
Use --explain to inspect source, category, risk, and confidence metadata.

            

          

          

            
            

              
Your data stays local

              
Scanning, planning, trashing, and restoring all run on your machine.

Commands you will use

        

          

            dev-cleaner scan
            
Scan the workspace and show a summary.

            
          

          

            dev-cleaner clean --dry-run
            
Preview what would be cleaned before deleting or trashing.

            
          

          

            dev-cleaner plan -o plan.json
            
Write a reviewable cleanup plan.

            
          

          

            dev-cleaner apply plan.json --trash
            
Apply a plan using undoable trash mode.

            
          

          

            dev-cleaner recommend --cleanup 10GB
            
Build a cleanup recommendation for a target amount of space.

            
          

          

            dev-cleaner undo
            
Restore the latest trash batch.
